Main actions:
  • LMB on node – select node
  • LMB on edge of selected node - select edge
  • Drag – move node (in normal mode) or create new edge (in edge editing mode)
  • Shift + LMB – multiselect node
  • RMB on ring scale – get filtering context menu
  • Shift + LMB on ring scale sector – show arcs that have one of their ends in this sector
  • Alt + LMB on ring scale sector – show arcs that have both their ends in this sector
  • Delete - remove selected edge
  • Backspace - remove selected edge
  • Alt + LMB on node - create new edge (in normal mode)
